# scribe: Command Argument Parsing

A base dependency solution with basic argument parsing for use with
'Rscript'.

## Details

For a quick overview, see
[`vignette("scribe")`](https://jmbarbone.github.io/scribe/articles/scribe.md)

## options

- `scribe.flag.no`:

  `[TRUE]`  
  Default value when `scribeArg` uses action `"flag"`

- `scribe.interactive`:

  `[NULL]`  
  Controls [`interactive()`](https://rdrr.io/r/base/interactive.html)
  check within internal `exit()`
